FILE LOOP

Descrizione: 

Esercizo per la creazione di un file in loop
	
Utilizzo:

In questo prormma andremo a creare un file in un percorso specificato, dato il nome, l'estenzione
e il contenuto del file dall'utente


Dipendenze:

Le librerie utilizzate in questo programma sono: /


Possibili Errori:

Controllare bene l'inserimento del path, in caso stiamo su sistema operativo Windows python non
riconoscera (\) come separatore del path, quindi va cambiato nel seguente modo (\\), inserendo 
due separatori invece di uno.
