TROVA FILE PER TIPO

Descrizione: 

Questo programma dato un path valido dell'utente contera quanti file ci sono all'intenro di quel path  
e di che tipo sono i file
	
Utilizzo:

Inserire un path valido, successivamente in automatico il programma ci dirà quanti file ci sono e di che tipo 
sono es: 3 file pdf, 4 file txt ecc.. 

Dipendenze:

Le librerie utilizzate in questo programma sono: os


Possibili Errori:

Controllare bene l'inserimento del path, in caso stiamo su sistema operativo Windows python non
riconoscera (\) come separatore del path, quindi va cambiato nel seguente modo (\\), inserendo 
due separatori invece di uno.

Note: 

Nell cartella è già presente una cartella chiamata "CartellaDiFile" dove è possibile fare delle prove, copiando 
quel path e inserendolo alla richiesta del programma
