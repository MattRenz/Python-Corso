SEGRETERIA SERVER V1


Descrizione:

Questo programma gestisce una segreteria lato server, in questo programma per esempio, dopo l'input dell'utetne di una segreteria come
nome, cognome, matricola, ed email, invia una stringa in un formato specifico: url ? contenuto. Questo programma ancora non salva i dati. 
Per impostare il programma inserire l'indirizzo ip e la porta del server da chiamare. Il programma risponderà 200 se la connesione con il
server è andata a buon fine e 500 se invece non è riuscito a comunicare con il server




Dipendenze (librerie):

Le librerie che usa questo progrmma sono: http.client / sys


Note:

Nella cartella FUNZIONI è presente la funzione SalvaSulServe, mentre nella cartella LIBRERIA c'è la gestione del server e di come deve rispondere (GET)

indirizzo IP = 127.0.0.1
Porta = 8081
