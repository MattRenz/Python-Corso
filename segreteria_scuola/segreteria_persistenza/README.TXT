SEGRETERIA PERSITENZA 


Descrizione:

Programma che salva tramite la persitenza, ovvero il concetto che ogni volta che eseguiremo il programma i dati che inseriamo veranno 
salvati all'interno di dei file. La clase figlie sono docenti e studenti, figli della classe persona, all'interno del file segrerteria 
sono presenti le funzioni per il salvataggio e la lettura del singolo studente e docente. All'intenro delle classi figli sono presenti 
alcuni metodi che permettono varie funzioni nel programma


Utilizzo:

All'inizio il progrmma ci chiederà di inserire in quale sezione vogliamo andare, docente o studente, una volta scelto la sezioni le funzionalità 
saranno analoghe. 

Studente: 

    1) Aggiungi studente 
    2) Inserisci voto 
    3) Calcolare media

Docenti: 

    1) Aggiungi docente
    2) Inserisci corso 
    3) Trova un corso


Dipendenze (librerie):

Le librerie che usa questo progrmma sono: smtplib / os / sys / re 


Note:

Il programma salvera le auto in questo path Windows: "C:\Users\nome utente che ha effetuato l'accesso\Programma_Segreteria"
Appena si esegue il prograamma si andranno a creare in automatico i DB dove saranno salvati i file per docenti, studenti, e password

Salvera le cartelle in automatico solo su sistema operativo Windows su sistema operativo Linux o altri bisognerà inserire il path di 
dove si vogliono salvare le cartelle.
